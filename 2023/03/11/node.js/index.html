基础窗口

命令行窗口 、cmd窗口

  • 1.开始菜单 –>运行 —> cmd—>回车

  • 2.常用指令：

    dir 列出当前目录下的所有文件

    md 目录名 创建一个文件夹

    cd 进入指定的目录

    rd 删除一个文件

  • 3.目录

    . 表示当前目录

    .. 表示上一级目录

  • 4.环境变量（）windows系统里的变量

    path

    C:\Program Files\Java\jdk-19\bin

  • 当我们在cmd中打开一个文件，或调用一个文件时。

    系统会首先在当前目录下寻找文件程序，如 果找到了则之间打开如果没有则会依次到path环境变量path中寻找，直到找到为止，如果没有就报错

所以我们可以将一些经常需要的文件的路径加入其中

进程和线程

  • 进程

    进程负责为车鞥徐的运行提供必备的环境

    相当于工厂的车间

  • 线程

    线程：计算机中的最小的计算单位，线程负责执行进程中的程序

    相当于工厂中的工人

  • 单线程

    js是单线程

  • 多线程

node.js

  • 简介

    node.js是一个能在服务器端运行javascript的开放源代码。跨平台javascript运行环境

    node采用v8引擎运行js代码，使用事件驱动、非阻塞、和异步I/O模型等来提高基础

  • node的服务器是单线程

    node是单线程但是后面有一个I/O线程池

模块化

  • CommonJs

    模块引用

    在Node中 require()函数来引用外部的模块

    require()可以传递一个文件的路径作为参数，node将自动根据该路径来引用外部模块

    模块定义

    模块标识

  • 模块化

    node中 每个js文件都是一个模块

    node中，每一个js文件的js代码都是独立运行在一个函数中，而不是全局作用域，所以一个模块化中的变量和函数在其他模块无法访问

  • npm（node package manager）

    对于node来说 npm帮助其完成了第三方模块的发布、安装和依赖等。借助npm Node与第三方模块之间形成了很好的一个生态系统

  • npm命令

    npm -v 查看npm的版本

    npm version 查看所有模块的版本

    npm search 包名 搜索包

    npm install/ i 包名 安装包

    npm remove/r 包名 删除包

    npm install 包名 –save 安装包并添加到依赖中

    npm install 下载当前依赖所需的包

  • node搜索包

    node在使用模块名字来引入模块时，它会当前目录的node_modules中寻找，若没有，择取上一级目录的node_modules中寻找，直到找到磁盘的跟目录，若没有则报错

  • buffer（缓冲区）

    buttfer的结构和数组很像，操作的方法也和数组类似
