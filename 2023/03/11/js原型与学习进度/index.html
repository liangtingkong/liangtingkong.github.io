js原型与原型链

js原型

  • 函数的prototype属性

    • 每个函数都有一个prototype属性，它默认指向一个object空对象（即为原型对象）

    • 原型对象有一个属性constructor，它指向函数对象

    • ` function fn (){}

              console.log(fn.prototype.constructor === fn)； ////true

      `

    给原型对象添加属性（一般是方法）==>实例对象引用

  • 显示原型属性与隐式原型属性

    函数中，都有一个prototype（显示原型属性），它指向函数的原型对象

    实例对象中，都存在一个__proto__(隐式原型属性)，它指向函数的原型对象

Object对象与Function函数 与Object.prototype

在了解原型与原型链之前，先简单理解一下Object对象 Function函数

object对象 ：在js中几乎所有的对象都是object对象的实例 由于这层关系，根据实例对象中的原型属性最终都可以找到Object.prototype

Function函数：简单来说每个 JavaScript 函数实际上都是一个 Function 对象

在根据原型的性质，所有的函数的隐式原型属性都指向Function.prototype
