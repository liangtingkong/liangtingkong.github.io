ES 6~11

为什么学习新特性

  1. 语法简洁，功能强大
  2. 框架开发应用
  3. 前端开发职位要求

let变量

  1. 变量不能重复声明

  2. 块级作用域:只在()/(代码块)里有用

  3. 不存在变量提升：不允许在变量之前使用变量

  4. 不影响作用域链：

声明常量 const i

  1. 一定要赋值
  2. 一般常量使用大写
  3. 常量值不能修改
  4. 块级作用域
  5. 对于数组和对象的元素修改，不算对常量的修改，不会报错

解构赋值(好处：可以直接调用)

  1. 数组解构

  2. 对象解构

    1. const zhao={
      name:’mqm’,
      age:’22’,
      fangfa:function(){
      log(‘xiaopin’);
      }
      }
      let {name,age,fangfa}=zhao;
      log(name);

es6简化对象

箭头函数 =>

特性
  • =>所调用函数中的this是静态的只会指向全局作用域中的那个元素

  • 不能作为构造实例化对象

  • 不能使用arguments变量

  • 箭头函数的简写

    • 形参只有一个的时候，可以省略小括号

    • 代码题只有一条的时候，可以省略花括号 此时return必须省略

总结
  • 箭头函数适合于this无关的回调，定时器，数组的方法回调
  • 不适合于this有关的回调，事件回调，对象的方法

参数赋初始值

  • 给形参的赋初始值（一般形参赋值给最后）
  • 与解构赋值结合使用

rest参数，获取函数实参，代替arguments

返回的为数组

rest参数必须要放到最后

    symbol基本使用

    symbol是新的数组类型，表示独一无二的值，是 js 中的第七种数组类型

    特点

    • 值是唯一，解决命名冲突

    • symbol值不能与其他数据进行运算

    • symbol定义的对象不能使用for…in遍历，但是可以使用Reflext.ownKeys来获取对象的所有键名

       let s1=Symbol('wang');
 console.log(s1);

 let s2=Symbol('wang')
 console.log(s2);
 console.log(s1===s2);

 //第二种方法   
 let s3 = Symbol.for('wang');
 let s4 = Symbol.for('wang');
 console.log(s3===s4);

第二种方法中s3===s4

注意事项
  • 不能运算对比

### 迭代器

迭代器（iteration）是一种接口，为不同的数据结构提供统一的访问机制

只有有iteration接口，就可以完成遍历操作

- es6有for...of循环     iteration接口主要使用for...of

- 原生具有iteration接口的数据有

  1. array
  2. arguments
  3. set
  4. map
  5. string
  6. typedarray
  7. nodelist

​ of 遍历的键值 in 遍历的键名

  • 工作原理（C语言链表相似）（迭代器用来达到自定义遍历）

    • 创建一个指针对象，指向当前的初始位置
    • 第一次调用对象的next方法，指针指向第一个成员
    • 不断调用next，指针自动指向下一个成员
    • 调用next方法返回一个value和done属性的对象

  • 应用

    通过添加迭代器来进行对自定义对象的遍历

    生成器

    一个特殊的函数 =>进行异步编程

class 类

MDN类

类是一种特殊的函数，定义类有两种方法，一种是类声明（类声明并不想函数声明一样会被提升），另一种是类表达式。

  • 类体和方法定义

    • 类中有一个唯一的constructor,用于创建和初始化 class创建的对象

    • 类中方法的添加是直接添加函数体

    • ```js

        class Person{
    constructor(name,age){
      this.name=name
      this.age=age
    }

    setName(name){
      this.name=name
    }
  }

  let s=new Person('mqm',25)
  s.setName('wangshuo')
  console.log(s);

  • 类的继承

    • 使用extends扩展子类

    • super()调用原函数 super()(用于访问和调用一个对象的父对象上的函数。)

    • ```js

       class Phone{
    constructor(brand,price){
      this.brand=brand
      this.price=price
    }

    call(){
      console.log('我可以打电话');
    }
  }

  class SmaartPhone extends Phone{
    constructor(brand,price,color,size){
      super(brand,price)

      this.size=size
    }

    photo(){

  }

  const oppo=new SmaartPhone ('oppo',1299,'黑色','11.inch')

数值扩展

  • Number.EPSILON=2.220446049250313e-16

    是js表示的最小精度

  • 二进制与八进制

  • Number.isFinite(num) 鉴定一个数值是否为有限数

  • Number.isNaN()判断一个函数是否为nan

  • Number.parseInt() 整数 Number.parseFloat() 浮点数 字符串转换为数字

  • Number.isInteger()检测一个数是否为整数

  • Math.trunc()将数字的小数部分抹去

  • Math.sign() 判断一个函数是正数(1) 负数(0) 还是零 (-1)

对象方法的扩展

  • Object.is(a,b)判断两个值是否相等

  • Object.assign(a,b) 对对象进行合并 若a对象与b对象有相同的属性名，其中a的属性值将被覆盖掉

模块化

将大的程序文件拆分成小的程序文件，然后将小文件组合起来

  • 模块化的好处

    • 防止命名冲突
    • 代码复用
    • 高维护性

  • 模块化规范

    • CommonJs=> nodeJs/browserify
    • AMD => requireJs
    • CMD => seaJs

  • 模块化语法

    模块功能主要有两个命令构成 export 和 import

    • export命令 用于规定模块的 对外 接口
    • import命令用于输入其他模块提供的功能

  • 暴露语法

    • ```js
      //分别暴露
      export let school=”htu”export function teach(){
      console.log('sb学校');
      }

import{school,teach} from  "../js/m1.js";

console.log(school);
console.log(xuexiao);
